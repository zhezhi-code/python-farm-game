# Python Farm Game

A desktop farm simulation game developed using **Python** and **Tkinter**, following the **Model–View–Controller (MVC)** architecture. The project demonstrates object-oriented programming, event-driven GUI development, inventory management, and interactive gameplay.

---

## Project Overview

This project is a simple farming simulation game where players can explore a farm, cultivate crops, manage inventory, trade items, and progress through multiple in-game days.

The project was developed to demonstrate software design principles, GUI programming, and object-oriented programming in Python.

---

## Features

- Interactive farm map
- Keyboard-controlled player movement
- Crop planting, harvesting and removal
- Inventory management
- Buy and sell system
- Energy, money and day progression
- Multiple map support
- Event-driven graphical interface

---

## Technologies

- Python 3
- Tkinter
- Object-Oriented Programming (OOP)
- MVC Architecture

## Gameplay

Players can:

- Move around the farm using the keyboard
- Till and until soil
- Plant different crops
- Harvest mature crops
- Buy and sell seeds
- Manage inventory
- Advance to the next day

---

## Controls

| Key | Action |
|-----|--------|
| W | Move Up |
| A | Move Left |
| S | Move Down |
| D | Move Right |
| T | Till Soil |
| U | Untill Soil |
| P | Plant Crop |
| H | Harvest Crop |
| R | Remove Crop |
